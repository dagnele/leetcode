# LeetCode Problem Solving Resources

## Knowledge

- [LeetCode 75 Study Plan](https://leetcode.com/studyplan/leetcode-75/)
  Curated 75-problem set for 1-3 months of preparation. Use for: the main problem pool because it is compact enough for a one-month, 30-minute/day plan.
- [LeetCode Top Interview 150](https://leetcode.com/studyplan/top-interview-150/)
  Larger classic problem set with broad topic coverage and editorials. Use for: extra practice after a pattern feels weak.
- [NeetCode Roadmap](https://neetcode.io/roadmap)
  Pattern-oriented roadmap for coding interview problems. Use for: deciding which pattern family a problem belongs to and finding adjacent drills.
- [CP-Algorithms](https://cp-algorithms.com/)
  High-quality algorithm explanations and reference implementations. Use for: deeper explanations of binary search, graphs, DP, data structures, and algorithmic invariants.
- [USACO Guide](https://usaco.guide/)
  Curated competitive-programming curriculum by strong practitioners. Use for: strengthening fundamentals when a LeetCode topic exposes a gap.

## Wisdom (Communities)

- [LeetCode Discuss](https://leetcode.com/discuss/)
  Community solution discussions. Use for: comparing approaches after you have already attempted a problem and written your own explanation.
- [USACO Forum](https://forum.usaco.guide/)
  Moderated programming-problem forum connected to the USACO Guide. Use for: asking conceptual algorithm questions when stuck beyond one problem.
